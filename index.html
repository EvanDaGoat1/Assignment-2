How to Apply CSS to HTML

To apply CSS to an HTML file, you can link an external CSS file using the <link> element within the <head> section of your HTML file. For example: 

        <head>
            <link rel="stylesheet" href="styles.css">
        </head>

How to Use JavaScript in a Webpage

You can include JavaScript in an HTML file in two ways:

  • Link an external JavaScript file using the <script> element:
    •         <script src="script.js"></script>
  • Write JavaScript directly within the HTML file using a <script> tag:
    •         <script>
            console.log("Hello, World!");
        </script>
